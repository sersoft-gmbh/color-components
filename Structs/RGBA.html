    Creates new RGBA components by splitting the combined hex value into RGBA components. E.g. RGBA<UInt64>(hex: 0x3D4A7C1B) becomes RGBA<UInt64>(red: 0x3D, green: 0x4A, blue: 0x7C, alpha: 0x1B).

  • init(hex:)

    Creates new RGBA components by splitting the combined hex value into RGBA components. E.g. RGBA<UInt8>(hex: 0x3D4A7C1B) becomes RGBA<UInt8>(red: 0x3D, green: 0x4A, blue: 0x7C, alpha: 0x1B). This overload exists so that e.g. RGBA<UInt8> can be created from 0xFFECDAFF which would overflow UInt8.

  • init(hexString:)

    Creates new RGBA components by parsing a given hex string into RGBA components. If the string has 0x or # as prefix, it is automatically removed. Returns nil if the string is no valid hex string. E.g. RGB<UInt8>(hexString: "0x3D4A7CFF") becomes RGB<UInt8>(red: 0x3D, green: 0x4A, blue: 0x7C, alpha: 0xFF).

  • hexValue()

    Computes the combined hex value for this RGBA components. E.g. for RGBA<UInt64>(red: 0x3D, green: 0x4A, blue: 0x7C, alpha: 0xFF) this returns 0x3D4A7CFF.

    Note

    If the Value of this RGBA components is too small to hold the entire hex value, this method will trap. Use hexValue(as:) to perform the convesion using a larger type.
